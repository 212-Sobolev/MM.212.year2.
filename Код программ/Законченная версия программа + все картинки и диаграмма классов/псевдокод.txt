
Инициализируем стартовый узел СТАРТ (стартовая точка);
СТАРТ.ставим метку 0;

Инициализируем конечный узел ГОАЛ (конечная точка);

Создаем приоритетную очередь ПО;
Добавляем к ПО СТАРТ;

пока(ПО не пусто и ГОАЛ не посещена) {
	Узел КАР = наибольший элемент ПО;
	Чистим наибольший элемент очереди;
	
	если (узел КАР уже посещен) {переходим к следующей итерации;}

	говорим что посетили узел КАР;

	для ("соседей" узла КАР) {
		КАРВЕС = значение метки в КАР + вес ребра от КАР к соседу;
		если (КАРВЕС < значение метки у соседа) {
			перезаписываем значение метки соседа как КАРВЕС;
			перезаписываем "предка" для соседа;
			добавляем соседа в ПО;
			}
	}
	
}

В обратном порядке возвращаясь от ГОАЛ по предкам до СТАРТ восстанавливаем наш путь.
